# 小红书图文转 PDF 项目

## 简介

本项目包含一个 Python 脚本 (`pdf.py`)，用于将从小红书批量下载的 ZIP 压缩包，智能地、按顺序地转换为 PDF 文件。

该脚本会自动处理 ZIP 文件，提取其中的图片，并根据图片的文件名进行自然排序，最后将它们合并成一个与原 ZIP 文件同名的 PDF。处理完成后，脚本会自动删除原始的 ZIP 文件以节省空间。

## 依赖

### 1. 浏览器扩展

您需要一个可以从小红书帖子批量下载图片并打包成 ZIP 文件的浏览器扩展。推荐使用：

-   **小地瓜 - 小红书图片视频下载助手**
    -   **Firefox 浏览器扩展**: [点击安装](https://addons.mozilla.org/zh-CN/firefox/addon/%E5%B0%8F%E5%9C%B0%E7%93%9C-%E5%B0%8F%E7%BA%A2%E4%B9%A6%E5%9B%BE%E7%89%87%E8%A7%86%E9%A2%91%E4%B8%8B%E8%BD%BD%E5%8A%A9%E6%89%8B/?utm_source=addons.mozilla.org&utm_medium=referral&utm_content=search)

### 2. Python 环境

请确保您的计算机上已安装 Python 环境。

### 3. Python 库

脚本依赖 `Pillow` 库来处理图片。如果您的环境中没有安装，请通过以下命令安装：

```bash
pip install Pillow
```

## 使用方法

### 模式一：批量处理（推荐）

1.  使用“小地瓜”扩展下载您需要的小红书图文，将所有下载的 `.zip` 文件移动到与 `pdf.py` 脚本相同的文件夹中。
2.  直接运行 `pdf.bat` 批处理文件，或者在命令行中运行 `python pdf.py`。
3.  脚本会自动查找当前目录下的所有 `.zip` 文件，并将它们一一转换为 PDF。

### 模式二：处理单个文件

您也可以通过拖放或命令行参数来处理单个文件：

1.  **拖放**：将一个 `.zip` 文件直接拖到 `pdf.bat` 图标上。
2.  **命令行**：打开命令行，运行 `python pdf.py "您的文件路径.zip"`。

脚本将只处理您指定的这一个文件。
